I'm a student in gymnasium "Jovan Jovanović Zmaj", web developer, linux programmer and a member of MasterBuilders team (also known as Makers). I'm developing in HTML5, JavaScript, Node.JS, PHP, CSS and other web app languages including Python3. In Makers NS we learn both hardware and software. We use microcomputers like Intel Galileo or Edison, Raspberry Pi and BeagleBone to learn how to use simple scripting languages like Python and Node.JS to control hardware of the microcomputers. All of the mentioned devices can be run as a linux machine so it is a great opportunity to learn linux and bash scripting.
